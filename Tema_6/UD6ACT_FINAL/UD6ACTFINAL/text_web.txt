Maquetem la web

En primer lloc has d'obrir l'entorn de desenvolupament preferit (Visual Studio, Php Storm o el que siga). Després has d'obrir un nou fitxer i emmagatzemar-lo com a html. El codi de baix mostra com fer la maquetació del primer "Hola món" 

    Obri l'editor
    Crea un nou fitxer html. En Visual Studio Code és amb les tecles ctrl + n
    Copia i pega el codi de baix
    Obri la pàgina en un navegador

                                

                                    

<div class="container">
    <div class="row">
        <div class="col-12">
        Hola món!
        </div>
    </div>
</div>
                                    
                                

                            

Publicar la web
Per publicar a la web és relativament senzill. Has d'utilitzar un client ftp com el filezilla i configurar-lo per accedir al teu hosting. Una vegada configurat només has de pujar els fitxers i ja està!!!
